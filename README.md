# Recipe Finder

Recipe Finder is a simple web application that allows users to search for recipes and view cooking instructions using data from TheMealDB API. The project is built with HTML, CSS, and vanilla JavaScript.

## Features

- Search recipes by name
- Display random recipes on the home page
- View recipe details in a modal
- Show ingredients and measurements
- Display cooking instructions
- Responsive user interface

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- TheMealDB API

## Getting Started

Move into the project directory:

```bash
cd recipe-finder
```

Open `index.html` in your preferred web browser.

No additional installation or dependencies are required.
